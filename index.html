Calculator info

A simple web calculator.
It rounds the answer to four decimal digits if the answer is longer than four decimal digits.
For full details visit the project on github.

Keyboard support

The calculator has keyboard support for all of the calculator buttons.
All keys match their button counterparts except for the following:

  • “Clear” is “Delete”
  • “=” is “Enter”
  • “⌫” is “Backspace”
