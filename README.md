# TOP Calculator
# Description
A simple web calculator.

# What it does
Preforms a mathematical operation based on the user selected inputs and displays the answer.

# How to use it
Open up the live preview by clicking [here]() (Link currently not available)
- Enter the first number you would like to use.
- Select the operator you would like to use.
- Enter the second number you would like to use.
- Select the equals button to preform a math operation using the first number, the second number, and the operator. Which will display the answer and reset the calculator.
- Or instead of selecting equals you could select another operator which will preform a math operation with the first and second numbers and the selected operator. Allowing you use the answer of the previous calculation and enter another number to preform a calculation on. You can continue do this until you select equals or clear.

Ex.
- Enter `15` First number is `15`.
- Select `+` Selected operator is `+`.
- Enter `5` Second number is `5`.
- Select `-` The operation `15 + 5 = 20` will be preformed, `20` will be displayed, `-` will be the selected operator, and `20` will be used as the first number in the next operation
- Enter `2` Second number is `2`.
- Select `=` The operation `20 + 2 = 22` will be preformed, `22` will be displayed and the calculator will be reset.

## Buttons
All of the buttons have keyboard equivalents which are the same as their button counterparts unless listed in parentheses.

### Digits:
`0-9` Enters a digit into the calculator.;

### Operators:
`+`  Selects addition. </br>
`-`  is subtraction. </br>
`*`  is multiplication. </br>
`/` is division.

### Other buttons:
`Clear` clears the display and resets the calculator (keyboard: `Delete`) </br>
`=` Equals, preforms the math operation, displays the results and resets the calculator. (keyboard: `Enter`) </br>
`.` Adds a decimal point. (You can only have one decimal point in a number). </br>
`⌫` is the backspace button, it removes the last entered digit or decimal on the display. (keyboard: `Backspace`)
